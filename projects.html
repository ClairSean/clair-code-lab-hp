我的项目

wincalc-web

WinCalc Web 是一个基于 Web 技术实现的 Windows 风格标准计算器，采用原生 HTML、CSS 和 JavaScript 开发。该项目支持数字输入（0-9）、小数点、正负号切换，以及加减乘除四则运算，同时提供清除（C）、退格（⌫）和等号（=）等基础功能，界面设计参考了 Windows 系统计算器的布局与风格。

项目源码托管于 GitHub，适合作为 Web 前端入门学习的参考案例。
